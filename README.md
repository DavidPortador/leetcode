# LeetCode Solutions by DP

This repository contains solutions to various LeetCode problems implemented in multiple programming languages.

## How to Run

To run the solutions, follow these steps:

1. Create the container **(first time only)**:

2. Navigate to the problem directory **(inside the container)**:
    ```bash
    cd n_problem_solutions/
    ```

3. Choose the language and execute the corresponding commands:

## Java
```bash
java solution.java
```
## Python3
```bash
python3 solution.py
```
## JavaScript
```bash
node solution.js
```
## PHP
```bash
php solution.php
```
## C#
```bash
dotnet solution.cs
```
## C++
```bash
cd cpp/
g++ solution.cpp
./a.out
```
## TypeScript
```bash
cd ts/
tsc solution.ts
node solution.js
```
